Quick Start

How to Use

  • Select your tuning at the top — changes apply immediately
  • Each session is 10 questions — takes 2–4 minutes
  • Harder items appear more often (spaced repetition)
  • Use keys 1–4 for multiple-choice answers
  • Progress is saved in your browser automatically

Note Memorization

Learn every note across the fretboard

Difficulty

Mode A — Find It

A note name is shown. Click all matching fret positions on the fretboard.

Mode B — Name It

A fret position is highlighted. Choose the correct note name from 4 options.

Tips

  • Start on Easy — all notes are shown so you can learn the layout by scanning
  • Move to Medium once you know the natural notes (A–G); use them as landmarks
  • Hard mode = full neck with zero hints — the real challenge
  • Octave shapes: the same note repeats diagonally — use that pattern
